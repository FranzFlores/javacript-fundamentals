Fetch

fetch es una función incorporada en JavaScript que se utiliza para realizar solicitudes de red y recuperar recursos (como archivos JSON, imágenes y otros datos) de un servidor web. Fetch se utiliza comúnmente en aplicaciones web para obtener datos de un servidor y actualizar la interfaz de usuario en función de los datos recuperados.
