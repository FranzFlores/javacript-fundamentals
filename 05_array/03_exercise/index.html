Ejercicio de arreglos usando JS

Realizar el código JS que cumpla con los siguientes requisitos:

  1. Pedir 6 números por pantalla (prompt) y se agreguen en un arreglo
  2. Mostrar el arreglo en la consola y en el body de la página
  3. Ordernar el arreglo y mostrarlo
  4. Mostrar la cantidad de elementos del arreglo
  5. Buscar un elemento ingresado por el usuario y responda si existe el elemento y su posición en el arreglo
