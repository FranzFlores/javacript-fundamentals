JSON

JSON es un acrónimo de "JavaScript Object Notation" y es un formato de intercambio de datos ligero y fácil de leer y escribir. En JavaScript, un objeto JSON es una cadena de caracteres que representa un objeto de datos estructurado. JSON es un formato muy comúnmente utilizado en aplicaciones web para enviar y recibir datos entre el cliente y el servidor, ya que es fácil de leer, escribir y analizar tanto por humanos como por máquinas.
