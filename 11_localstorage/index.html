LocalStorage

LocalStorage es una característica del objeto window en JavaScript que permite almacenar y recuperar datos de manera persistente en el navegador web. Es un tipo de almacenamiento local de datos que se puede utilizar para almacenar información que el usuario ha ingresado en un formulario o para almacenar la configuración del usuario en una aplicación web.
